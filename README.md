# 🪙 Galleon

**极致简约、隐私优先的个人财务航海日志。**

> "Less, but better." — Dieter Rams

Galleon 是一款基于 **本地优先 (Local-first)** 理念构建的个人财务管理 PWA。它不只是一个账本，更是你财务世界的个人仪表盘。

---

## ✨ 核心亮点

### 🔐 绝对隐私，归还控制权
你的数据永远属于你。没有账号，没有服务器，没有追踪。所有财务流水仅存储在浏览器本地的 IndexedDB 中，绝不离开你的设备。

### 🧠 自然语言，极速记账
告别繁琐的选择和填表。直接输入 “今天星巴克 35” 或 “打车 42.5 元”，Galleon 强大的本地解析引擎会瞬间识别分类、金额和商家，全程无需联网。

### 🎨 迪特·拉姆斯美学
遵循“少即是多”的设计原则。金银色调的精密质感，搭配 Playfair Display 与 Inter 字体，在光影流转中提供丝滑、克制且高级的财务记录体验。

### 📲 离线使用，无处不在
作为一款现代 PWA，Galleon 支持安装到主屏幕。无论是有信号还是身处深海，应用都能秒开，记账从未如此流畅。

---

## 🚀 立即开始

### 预置要求
- Node.js 18+
- NPM / PNPM / Bun

### 快速启动
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
然后在浏览器中打开 [localhost:3000](http://localhost:3000) 即可开始航行。

---

## 🛠️ 技术基石

Galleon 建立在最先进保持 Web 技术之上，旨在提供原生般的性能：

- **核心架构**: Next.js (App Router) + React 19
- **本地数据库**: Dexie.js v4 (基于高级索引的 IndexedDB 管理)
- **视觉风格**: Tailwind CSS 4 + Shadcn UI
- **动效反馈**: Framer Motion (包含标志性的硬币掉落动态与物理效果)
- **数据安全**: Zod 运行时强类型校验

## 🤝 隐私声明

Galleon 坚信财务隐私是基本权利。

- **零遥测**: 我们不收集任何匿名分析数据。
- **透明性**: 所有的逻辑运行在你的客户端，代码保持开放透明。
- **可移植性**: 你的数据永远可以导出为 CSV 或 JSON，不会被任何生态锁死。
